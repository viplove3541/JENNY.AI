Features: Understand and response to natural language, It can write codes and help developers in solving coding-related problems.

Tech Stack Used: HTML,CSS,Javascript,Vite (Bundler),Openai (API),Cors (Middleware),Dotenv (Environment Variables) & Express (Backend).

Description: The goal of this project is to create a chatbot that can talk to you about programming. The chatbot was developed with the OpenAI API, which uses GPT-3 to generate responses to messages sent by the user.
